## Build Setup

``` bash
# install dependencies
npm install
yarn 

npm run dev

npm run build

```

```
api--> xhr api
assest--> images
components --> 组件
	下面对应所在网页，公用建立公用文件
pages --> 页面所在 当前目录只能建立index文件夹
	pages->index->router-->当前页面router所在,
	js 必须有_main.js结尾。
	前缀一样
styles--> 样式表
utils--> js
vuex--> 状态管理目录
view-->组件view
新建页面重新启动
```
